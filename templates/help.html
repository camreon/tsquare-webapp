Help Documentation

Welcome to the new T-square!

Getting Started

If you already have a T-square account, you can log in using your existing credentials by visiting http://tsquare-webapp-temp.herokuapp.com

Mobile/Web

The new T-square is mobile/web and should scale appropriately on any mobile browser.

Navigation

On the desktop version, the navigation bar will be at the top. However, in the mobile version, the navigation bar can be accessed by clicking a button all the way at the top on the right.

  • Tools

    Here you can find a list of links relative to each of your courses.

    • Course Info
    • Announcements
    • Assignments
    • Resources
    • Gradebook
    • Wiki

  • Courses

    Here you can find a list of all the courses you're taking this semester.

  • Profile Settings

    Here you can find options to set up some useful features for your account, such as Notifications, Time Zone, Integrating with external services etc.

Viewing Courses

To view all your courses, click on 'Courses' at the top

Integrating Account With External Services

You can click on Profile Settings at the top and then click Services to view which services you can integrate your account with.

  • Google Drive

    Integrating with Google Drive allows you to pick which files from your Google Drive would you like to include in an assignment submission

  • Github

    Integrating with Github allows you to attach code from repositories for submitting code to a programming assignment.
